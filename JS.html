Introduction to JavaScript

JavaScript or JS is a interpreted programming language which are used in HTML to create more dynamic and convenience to the user as well as the coder.

It is not unusual for beginner programmer to associate JavaScript with Java, since my my first code was Java I also assumed that it would be very closely related, but that is not true.

While there are similarity with loops, conditional, variable storage, the way JavaScript operate are different from Java. For once Java can only be operate on a virtual machine while as JavaScript can only be run on browsers.

JavaScript is definitely a wonder for HTML file, it added dynamic to the site with its method and function (a subprogram).

The block of codes in JavaScript can be organized neatly and conveniently for programmer to assign specific task for the function.
