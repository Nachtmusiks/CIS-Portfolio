Introduction to SQL

Perl is a general purpose programming language originally developed for text manipulation, but are now used widely for tasks such as system administration, network programming and more importantly web development.

If SQL is the bridge between the user and the programmer, then Perl is the building block for that bridge.

For this particular course we learn Perl instead of PHP due to security reason. My experience with Perl has been complicated, it is quite unlike most programming language I have been introduced to and I do think that it is the most complicated in this course.

The usage of Perl in web development seems to revolve around connecting the user input data and transferring it appropriately to the database. The process sounds simple but it require complicated maneuver to manipulate both SQL and and Perl and incorporate it together to achieve.

There are several resources on this page that can guide you to learning Perl. There are 3 powerpoints available explaining server side scripting, connecting to database and how to retrieve as well as a Youtube playlist of Perl learning.

On the assignment columns I displayed sending and retrieving data back from the database using both SQL and Perl. There is also a form example contained with the iFrame to demonstrate it further.
