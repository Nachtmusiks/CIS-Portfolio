Introduction to SQL

SQL stands for Structured Query Language and it is widely used to store and communicate data that is input by the users.

As part of learning goal in Web Development class this language is crucial to bridge the gap between the user and the server. The language itself is fairly straight-forward and are very much case sensitive.

It is mainly used to manage the data that is input by the users. These data can be generate from HTML client side and be send to the database you created.

These data are then modify or store in the database by the creator. These data are only able to be store in a structured table.

On the assignment column shows my first database table with each column specifying type of user's data. It also included the server side script example with the SQL conditional state to show how you can search through your data.
