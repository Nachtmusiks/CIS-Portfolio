Introduction to HTML and CSS

HTML and CSS are markup languages that are used to create webpages. These websites are what connecting users all over the world together and allowing the spread of information to be more accessible and fast.

I previously started out with Java first, so I noticed that HTML are fairly easy due to it relatively straight-forwardness. CSS comes much easier too because the structures remind me of Java.

My first learning of HTML and CSS started out at Codecademy, but I also use HTML & CSS book by Jon Duckett to further my understanding.

In addition to that I also use W3 Schools, Stack Overflow, and CSS as resources to design a more efficient website. Color Hexa is a also a great website to finding the approriate color theme for webpages.

After learning the basic of HTML and CSS I began creating webpages on CPanel. A hosting site for webcreators.
